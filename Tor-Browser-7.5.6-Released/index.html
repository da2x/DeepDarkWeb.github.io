Tor Browser 7.5.6 Released

The Tor Project released Tor Browser Version 7.5.6.

Tor Browser Version 7.5.6 features important security updates to Firefox.

In addition, Tor Browser 7.5.6 updates Firefox to 52.9.0esr and includes newer versions of NoScript and HTTPS Everywhere and the latest Tor stable version, 0.3.3.7.

This Tor Browser version also contains a number of backported patches from the alpha, most notably the feature to treat cookies set by .onion domain as secure as well.

For Windows users the Tor Project activated an option that prevents an accidental proxy bypass when dealing with UNC paths.
